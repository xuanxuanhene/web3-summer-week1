| 活动时长 | 60–75 分钟 |

---

## 一、活动主题

**「AI Agent 零基础入门：从概念到第一个可用智能体」** —— 这是一场专注于 **认知破除与行动启动** 的线上 Space。

主题选择 AI Agent 而非泛化的 AI 或 Web3，是因为 AI Agent 恰好处于 **AI 与 Builder 的交汇点**：它既是当下技术趋势中最具爆发力的方向，又是新人最可以「动手做出东西」的切入点。2026 年，AI Agent 已从概念验证走向规模化落地，Mastra、LangChain、CrewAI 等框架大幅降低了开发门槛，无代码平台（如 Fleece AI、Zapier AI）进一步让非技术人员也能参与构建。[^1]

---

## 二、活动目标

### 三大核心目标

1. **认知破除** — 打破"AI Agent 需要博士学历"的认知壁垒
2. **行动启动** — 提供清晰可复现的零基础入门路径
3. **社群连接** — 建立新人互助的轻量学习社群

### 具体目标

| # | 目标 | 说明 |
|---|------|------|
| 1 | 打破「AI Agent 需要博士学历」的认知壁垒 | 通过真实案例展示：一个会用搜索引擎的人，就能在 30 分钟内搭建第一个 AI Agent。消除「我不够格」的心理障碍。 |
| 2 | 提供一条清晰、可复现的零基础入门路径 | 给出一份「从 0 到 1」的路线图，包括工具推荐、学习资源、社区入口，让参与者离开时知道自己下一步该做什么。 |
| 3 | 建立新人互助的轻量社群 | 通过 Space 的互动问答环节和后续社群链接，让参与者在活动后能找到同伴、继续交流，降低独自摸索的孤独感。 |
| 4 | 桥接 Web3 与 AI 两个社区 | Web3 社区中有大量对 AI 感兴趣但不知如何入门的 Builder，AI 社区中也有对去中心化好奇的开发者。这场 Space 成为两个群体的交汇点。 |

## 三、目标听众

本场 Space 的核心听众画像如下。不做「所有人」的活动——**精确的听众定义是活动效果的前提**。

| 听众画像 | 典型特征 | 核心痛点 | 为什么邀请他们 |
|----------|----------|----------|----------------|
| **Web3 新手 Builder** | 刚加入 Web3 社区 1–6 个月，对智能合约有初步了解，想扩展 AI 技能 | 知道 AI Agent 很火，但不知道从哪里开始，怕自己技术不够 | 他们已有 Builder 思维，缺的是 AI 方向的「第一推动力」 |
| **AI 好奇者 / 转行者** | 传统行业从业者（产品、运营、设计、市场），对 AI 感兴趣但无编程背景 | 大量碎片化信息带来焦虑，不知道哪些是真正值得投入时间的 | 他们代表了最广泛的「门外人群」，帮助他们入门是扩大社区影响力的关键 |
| **Web3 社区运营者** | 负责社区增长、内容、活动的运营人员 | 需要了解 AI 趋势以策划社区内容，但自己没有深度使用过 AI 工具 | 他们能成为「放大器」——在自己的社区中传播 AI Agent 的入门知识 |
| **学生 / 早期职业探索者** | 在校大学生或毕业 1–2 年，正在探索职业方向 | 看到 AI Agent 开发者的高薪机会，但不确定自己是否适合这条路 | 他们是最有学习动力、最需要方向指引的群体 |

> **关键原则：** 不假设参与者有任何编程经验。所有讨论、演示和推荐资源都从「零基础」出发。如果一场 Space 让零基础的人感到被排斥，那它就失败了。

## 四、核心问题

这场 Space 希望解决的不是一个技术问题，而是 **一组层层递进的认知与行动障碍**。以下是梳理的核心问题链：

### 1. 「AI Agent 到底是什么？和 ChatGPT 有什么区别？」

新人最基础的困惑——概念不清。AI Agent 不是聊天机器人，而是能够自主规划、调用工具、执行多步任务的智能体。用 5 分钟讲清楚这个区别，是整场活动的认知基石。

### 2. 「我需要学多少编程才能开始？」

这是阻隔最多人的心理门槛。事实上，2026 年的无代码/低代码平台已经让零编程基础的人也能搭建 Agent。需要明确展示三条路径：**无代码 → 低代码 → 代码框架**，让每个人找到适合自己的起点。

### 3. 「我第一次应该做什么？有没有一个最小的可完成项目？」

新人最怕的不是「难」，而是「不知道做什么」。需要一个具体的、可在 30 分钟内完成的「第一个 Agent」项目作为起点——例如「一个自动整理每日 Web3 行业新闻并推送到 Telegram 的 Agent」。

### 4. 「我学了这个能做什么？有什么实际价值？」

需要展示 AI Agent 在真实场景中的应用：自动处理客户支持、链上数据分析、自动化内容创作、DeFi 策略执行等。让新人看到「这个技能有真实的市场需求」。

### 5. 「我该去哪里找到一起学习的人？」

独自学习的放弃率极高。这场 Space 本身就是一个「社群入口」——通过活动建立联系，让参与者找到学习伙伴，形成持续的正反馈循环。

## 五、为什么这个主题值得举办

### 1. 时机窗口：2026 年是 AI Agent 的「大众化元年」

2026 年，AI Agent 生态已经成熟到足以让非技术人员参与构建。微软推出了面向新手的 Agent-a-thon 系列活动，Mastra 等框架提供免费线上 Workshop 教学如何「构建可持续运行数天的 Agent」[^2]。与此同时，主流 AI Agent 开发框架（LangChain、CrewAI、AutoGen）和无代码平台（Fleece AI、Zapier AI、Botpress）已形成完整生态[^1]。这意味着 **现在是新人入场的黄金窗口**——既有成熟工具降低门槛，又还未到「人人都会」的红海阶段。

### 2. 需求真实：大量新人在门外徘徊但缺乏引导

当前中文互联网上关于 AI Agent 的内容两极分化严重——要么是学术论文级别的深度技术分析，要么是「AI 将取代一切」的焦虑贩卖。真正面向零基础、提供可操作入门路径的中文内容严重不足。TinTinLand 等 Web3 开发社区的 AI Agent 训练营报名火爆（单期超过 300 人），说明需求真实且迫切[^3]。

### 3. 社区价值：Space 是新人最友好的入门形式

相比训练营（需要数周时间承诺）和文档（需要自学能力），Twitter Space 的 **低门槛、实时互动、社交属性** 使其成为新人入门的最佳场景。一场 60 分钟的 Space 足够让参与者建立基本认知、获得行动方向，并产生「我也能做到」的信心。根据社区运营实践，AMA 和 Space 形式在 Web3 社区中承担着「项目冷启动与开发者招募的第一入口」功能[^4]。

### 4. 战略卡位：AI Agent 是 Web3 的下一个增长引擎

AI Agent 与 Web3 的结合正在催生新赛道：去中心化 AI Agent、链上自主交易 Agent、AI 驱动的 DAO 治理等。谁能率先在社区中培养一批懂 AI Agent 的 Builder，谁就能在下一波创新浪潮中占据先机。这场 Space 不仅是一次教育活动，更是 **社区人才储备的战略投资**。

> 「Agent 不是『装上就能用』，而是『会用才有价值』。」—— 网易智企 Agent 实战特训营的核心理念，也正是本场 Space 希望传达的：从「知道」到「做到」[^5]。

## 六、参考案例

### 参考案例 1：Microsoft AI Power Days: Agent-a-thon（2026 年 2 月）

- **来源：** Microsoft Tech Community[^2]
- **简介：** 微软于 2026 年 2 月举办的 Agent-a-thon 活动，面向不同技术水平的参与者设置了分层轨道——从无代码基础到复杂多 Agent 系统。活动的核心理念是「Choose your track, and learn to build agents your way」，充分体现了分层教学、因材施教的思路。
- **借鉴点：** 分层设计——在 Space 中可设置「无代码入门」「低代码进阶」「代码框架」三条路径的简要介绍，让不同背景的参与者都能找到适合自己的起点。同时，微软的「Agent-a-thon」命名方式（Agent + Marathon）强调了「动手实践」的定位，值得学习。

### 参考案例 2：Mastra Workshops: Build Durable Agents That Run for Days（2026 年 7 月）

- **来源：** Mastra 官方 Workshops[^6]
- **简介：** Mastra 推出的免费线上直播 Workshop 系列，由团队成员和外部专家主持，面向想要学习构建和部署 AI Agent 的开发者。活动提供录制回放和代码示例，注册即可参与。该系列每周举办，已形成稳定的社区学习节奏。
- **借鉴点：** （1）免费 + 录制回放的模式降低了参与门槛；（2）每周固定节奏培养了社区习惯；（3）提供可复用的代码示例，让参与者活动后能立刻动手。本场 Space 可参考其「提供可带走的学习资料」的做法，在活动结束后分享一份精简的入门资源包。

### 参考案例 3：TinTinLand × 0G/Flow/Near/Phala: AI Agent 核心技术开发入门训练营（2025 年）

- **来源：** TinTinLand 社区[^3]
- **简介：** TinTinLand 联合多条公链推出的 4 周在线训练营，涵盖 AI Agent、智能合约开发、区块链应用等主题，每周 2–4 次在线技术分享与互动讨论，超过 300 名开发者参与。活动成功的关键在于将 AI Agent 与 Web3 生态紧密结合，展示了跨领域学习的实际价值。
- **借鉴点：** （1）AI Agent 与 Web3 的结合是吸引社区参与的核心卖点；（2）短周期（4 周）、高频率（每周多次）的节奏保持了学习动力；（3）多链联合的方式扩大了受众覆盖面。本场 Space 作为「轻量级入口」，可以为后续深度训练营做预热和引流。
